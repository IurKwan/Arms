Class PlayerConfig

  • 
public final class PlayerConfig

    Supplies the classes necessary for the com.devbrackets.android.exomedia.nmp.ExoMediaPlayerImpl to play media. It is recommended to use the PlayerConfigBuilder to construct a new instance as that will provide default implementations.
