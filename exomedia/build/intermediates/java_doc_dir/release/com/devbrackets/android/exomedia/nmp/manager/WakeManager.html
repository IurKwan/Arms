      • setWakeLevel

         final Unit setWakeLevel(Integer levelAndFlags)

        This function has the MediaPlayer access the low-level power manager service to control the device's power usage while playing is occurring. The parameter is a combination of android.os.PowerManager wake flags.

        Use of this method requires android.Manifest.permission.WAKE_LOCK permission.

        By default, no attempt is made to keep the device awake during playback.

        Parameters:
        levelAndFlags - The wake lock level and any flags to apply, see PowerManager.newWakeLock

      • stayAwake

         final Unit stayAwake(Boolean awake)

        Used with playback state changes to correctly acquire and release the wakelock if the user has enabled it with setWakeLevel. If the wakeLock is null then no action will be performed.

        Parameters:
        awake - True if the wakelock should be acquired
