      • setWakeLevel

         Unit setWakeLevel(Integer levelAndFlags)

        Set the low-level power management behavior for this AudioPlayer.

        This function has the AudioPlayer access the low-level power manager service to control the device's power usage while playing is occurring. The parameter is a combination of android.os.PowerManager wake flags. Use of this method requires android.Manifest.permission.WAKE_LOCK permission. By default, no attempt is made to keep the device awake during playback.

        Parameters:
        levelAndFlags - the power/wake mode to set
