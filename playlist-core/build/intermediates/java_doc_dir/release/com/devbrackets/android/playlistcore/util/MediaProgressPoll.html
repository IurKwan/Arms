Class MediaProgressPoll

  • 
public class MediaProgressPoll<I extends PlaylistItem>

    A utility used to poll the progress of the currently playing media. This will allows listeners to be informed of progress updates for display or storage.

      • stop

      • reset

         final Unit reset()

        Stops the current progress poll and clears out the previously set values with .setOverridePosition, .setOverrideDuration, and .setOverriddenDuration

      • release

         final Unit release()

        Releases all resources associated with the poll including the media player specified with .update and the progress listener specified with .setProgressListener.

        NOTE: this calls .reset

      • update

      • setProgressPollDelay

         final Unit setProgressPollDelay(Integer milliSeconds)

        Parameters:
        milliSeconds - The millisecond delay to use

      • restartOverridePosition

         final Unit restartOverridePosition()

        Restarts the audio position to the start if the position is being overridden (see .setOverridePosition). This will be the value specified with .setPositionOffset or 0 if it hasn't been set.
